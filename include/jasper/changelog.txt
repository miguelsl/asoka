jasper-4_5_1_2 (28/08/2012)
- Se actualizan los drivers de Oracle (fichero classes12.jar) a los de la versión 10.2 (última versión compatible con
Oracle 8)
- Cuando el informe se lanza contra una base de datos Oracle a través de conexión OCI8, según versiones es necesario
que la variable de entorno NLS_LANG esté definida, se modifica el código para comprobar que dicha variable tiene valor
y de no ser así se fija a SPANISH_SPAIN.WE8ISO8859P15. Si se desea otro valor, basta con fijar en el código PHP a través
de putenv("NLS_LANG=IDIOMA_PAIS.CODIFICACION") antes de invocar al método createResultFile().
